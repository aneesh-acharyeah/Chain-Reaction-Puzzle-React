# Chain Reaction Puzzle

A strategic bomb placement puzzle game built with React. Place bombs on a grid to trigger explosive chain reactions that clear the board. Plan your moves carefully to maximize cascades and win!

## Features

- 6x6 grid with limited bombs
- Place bombs and trigger chain reactions
- Explosions affect adjacent cells, causing cascades
- Quick paced and strategic gameplay
- Clear the board to win
- Simple, intuitive UI with visual bomb/ explosion feedback
